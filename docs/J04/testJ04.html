    Q-11: Како ће изгледати низ L након извршења следећег програмчића:

    L = [5, 1, 6, 2, 3, 0]
L.sort()
  • [5, 1, 6, 2, 3, 0], тј. ништа се неће променити.
  • Покушај поново!
  • [0, 1, 2, 3, 5, 6], тј. елементи низа ће бити поређани од мањих ка већим вредностима.
  • Тачно!
  • [6, 5, 3, 2, 1, 0], тј. елементи низа ће бити поређани од већих ка мањим вредностима.
  • Покушај поново!
  • Систем ће пријавити грешку јер функција sort захтева додатни аргумент.
  • Покушај поново!

    Q-12: Како ће изгледати низ L након извршења следећег програмчића:

    L = [5, 1, 6, 2, 3, 0]
L.sort(reverse=True)
  • [5, 1, 6, 2, 3, 0], тј. ништа се неће променити.
  • Покушај поново!
  • [0, 1, 2, 3, 5, 6], тј. елементи низа ће бити поређани од мањих ка већим вредностима.
  • Покушај поново!
  • [6, 5, 3, 2, 1, 0], тј. елементи низа ће бити поређани од већих ка мањим вредностима.
  • Тачно!
  • Систем ће пријавити грешку јер функција sort не сме да садржи додатни аргумент.
  • Покушај поново!

    Q-13: Који од понуђених Пајтон израза из низа L = [5, 1, 6, 2, 3, 0] издваја све парне бројеве?

  • [x for x in L if x % 2 == 0]
  • Тачно!
  • [x for x in L if x % 2 != 0]
  • Покушај поново!
  • [x for x in L and x % 2 == 0]
  • Покушај поново!
  • [x : x in L and x % 2 != 0]
  • Покушај поново!

    Q-14: Који од понуђених Пајтон израза из низа L који је наведен испод издваја све стрингове чија дужина је 12 или више?
